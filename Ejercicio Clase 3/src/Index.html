¿Qué es HTML5?

HTML5 es la última versión del lenguaje de marcado HTML. Introduce nuevas etiquetas, atributos y funcionalidades que mejoran la estructura semántica, el soporte multimedia y la experiencia de usuario.

Nuevos elementos semánticos

HTML5 introduce nuevas etiquetas como <section>, <article>, <nav> y <aside> que ayudan a estructurar el contenido de forma más clara y significativa.

Soporte para multimedia

HTML5 facilita la incorporación de audio y video con las etiquetas <audio> y <video>, sin necesidad de plugins adicionales.

Accesibilidad y compatibilidad

Gracias a su estructura semántica, HTML5 mejora la accesibilidad y la compatibilidad con distintos dispositivos, haciendo que la web sea más universal.
